Welcome to JSadaKhata

JSadakhata is a Javascript+HTML implemention of SADAKHATA. You just need to Download->unzip->Open sadakhata.html. It'll automatically opened with your default browser. Then write :)

For Developer

If you are a Developer(#respect) you can use this tool easily!!! In your HTML page, add these lines to in <head></head>

<link rel="stylesheet" href="CSS/jquery-ui.css" />
<script type="text/javascript" src="JS/jquery.min.js"></script>
<script type="text/javascript" src="JS/jquery-ui.min.js"></script>
<script type="text/javascript" src="JS/sadakhata.js"></script>
<script type="text/javascript" src="JS/jsDB.js"></script>
<script type="text/javascript" src="JS/avro.js"></script>
<script type="text/javascript" src="JS/jquery.caret.1.02.js"></script>

Note: Keep the files in perfect directory.

now Let You have a textarea/input field with id="inp". Then to Active sadakhata to this field add these lines in <head></head>

$(function() {
    sadakhata($("#inp"));
});
